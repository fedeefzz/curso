Ejemplo de selectores descendientes

Las etiquetasLI son descendientes de la etiqueta UL. La etiqueta A es descendiente de las etiquetas LI (descendiente de un hijo) y UL (descendiente de un nieto).>


Si piensas en esto usando el ejemplo de un árbol genealógico, el hijo de ese elemento sería el padre, el hijo de ese elemento, y el nieto del nieto.

Entonces, ¿cómo dirigirías elementos específicos en una página web utilizando estos selectores de descendientes? En primer lugar, se deben definir los selectores de descendientes utilizando dos (o más) selectores de tipo separados por espacios:

 li a {
        color:brown;
        background-color: bisque;
        font-size: 20px;
    }
